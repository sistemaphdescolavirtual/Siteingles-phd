# Operação — PHD Escola Virtual

## 1. Perfis do sistema

### Gestor

Pode:

- Ver professores
- Ver alunos
- Ver atividades
- Ver respostas
- Ver anexos
- Vetar usuários
- Reativar usuários
- Cadastrar novos gestores

### Professor

Pode:

- Ver alunos vinculados
- Criar atividades
- Enviar anexos
- Corrigir respostas
- Enviar mensagens
- Receber notificações

### Aluno

Pode:

- Ver atividades
- Abrir anexos
- Enviar respostas
- Conversar com professor
- Receber notificações

## 2. Fluxo de aluno

1. Aluno se cadastra.
2. Informa código do professor.
3. Cadastro fica pendente.
4. Professor aprova.
5. Aluno consegue acessar.

## 3. Fluxo de atividade

1. Professor cria atividade.
2. Aluno recebe notificação.
3. Aluno responde.
4. Professor recebe notificação.
5. Professor corrige.
6. Aluno recebe retorno.

## 4. Fluxo de chat

1. Professor abre conversa com aluno.
2. Envia mensagem.
3. Aluno recebe notificação.
4. Aluno responde.
5. Professor acompanha pelo painel.

## 5. Fluxo do gestor

1. Gestor acessa painel.
2. Analisa usuários.
3. Pode vetar ou reativar.
4. Pode cadastrar novos gestores.
5. Pode visualizar atividades e anexos.
